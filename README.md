# set-high_level_programming

This repository contains projects from the High-Level Programming track,
covering Python scripting fundamentals alongside continued C practice
through technical interview preparation exercises.

## Projects

### python-hello_world

An introduction to Python: running scripts from the shell, printing with
`print()` and f-strings, string indexing and slicing, standard streams,
byte-compiling `.py` files, reading Python bytecode, and a Floyd's
cycle-detection exercise in C.
