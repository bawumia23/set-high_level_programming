# Python - Data Structures

This project is part of the ALX Software Engineering curriculum.

## Description

This project introduces Python data structures, including:
- Lists
- Tuples
- Strings
- List manipulation
- Matrix operations
- Basic algorithms on lists

All solutions are written in Python 3.8.5 and follow the pycodestyle (PEP 8) style guide.
