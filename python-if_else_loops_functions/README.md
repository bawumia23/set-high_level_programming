# 0x01. Python - if/else, loops, functions

## Description

This project covers the basics of Python programming: conditional
statements, loops, and functions. It also includes a C exercise on
inserting a value into a sorted singly linked list.

## Learning Objectives

- Why Python programming is awesome
- How to use the `if`, `if ... else` statements
- How to use comments in Python
- How to assign values to variables
- How to use the `while` and `for` loops
- How to use `break` and `continue` statements in loops
- How to use `else` clauses on loops
- How to use `range` function
- What is a function and how to use functions in Python
- What are the default values of parameters and how to use them
- What is variable scope (local vs global)
- How to write a C function that inserts a number into a sorted
  singly linked list

## Requirements

### Python Scripts

- Ubuntu 20.04 LTS, Python 3.8.5
- Style checked with `pycodestyle` (version 2.8.*)
- All files executable, ending with a new line
- First line of all files: `#!/usr/bin/python3`

### C Scripts

- Compiled on Ubuntu 20.04 LTS with `gcc`, using the flags:
  `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Follows the Betty coding style
- All header files include guarded
